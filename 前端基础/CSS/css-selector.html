css选择器

1.基本选择器

1.1 通配符选择器：通配符选择器常用‘*’号表示,它是所有选择器里作用范围最广的,能匹配页面中所有的元素

1.2 元素选择器：tag名称，比如"h1"

1.3 id选择器：“#”

1.4 类选择器：“.”

1.5 属性选择器："tag[attr="value"]"

1.6 伪类选择器：比如“:first-child”

1.7 伪元素选择器：比如“:before”,增加“content: "*"”

2.组合选择器

2.1 后代选择器：“body h1” 支持多层级

2.2 子代选择器：“body>h1”仅子层级。子代必是后代，后代不一定是子代。

2.3 相邻兄弟选择器：“.brother+p”，元素紧邻的兄弟节点

2.4 通用兄弟选择器：“.brother2~h2”，元素后所有满足选择条件的兄弟节点
