1.数据类型检测

1.1 typeof: 基本类型 string、number、boolean、undefined、symbol外，其余绝大多数都是object。例外：function为function，null也是object

1.2 instanceof：原理为利用原型及原型链判断实例对象与构造函数之间的关系。只能判断object，基础类型无法判断

1.3 Object.prototype.toString.call()。 返回结果是 [object xxx]，因类型而异
